东周: Eastern Zhou

秦兴师临周而求九鼎

秦興師臨周而求九鼎， 秦国义兵临近周国去要九鼎， 周君患之，以告顏率。 周王担心秦军，所以告诉颜率。颜率，人名。 顏率曰：「大王勿憂，臣請東借救於齊。」 颜率说「大王勿忧郁，您臣会往东请叫齐国的借助来救我们。」 顏率至齊，謂齊王曰： 颜率到了齐国，请叫齐王说 「夫秦之為無道也， 「那个秦国的行为一点道理者没有， 欲興兵臨周而求九鼎， 想义兵来到周而拿九鼎， 周之君臣，內自盡計， 我周王的臣，我已经什么计划都耗尽了， 與秦，不若歸之大國。 跟秦，不如归回大国。—— 所谓大国应该指周朝。 夫存危國，美名也； 如果他们还存在，就对您国、您美名危险， 得九鼎，厚寶也。 得到九鼎，是得到个珍贵之宝。 願大王圖之。」 我想大王参议于此。 齊王大悅，發師五萬人， 齐王非常高兴，发布师军数五万人， 使陳臣思將以救周，而秦兵罷。 使陈臣思领而去解救周国，而罢秦兵。—— 陈臣思：人名。罢繁体罷。
